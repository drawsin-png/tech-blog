## 🎯 오늘 목표했던 것
- 운전자 졸음 감지(Drowsiness Detection) AI 모델 개발 및 테스트

## 🚀 새롭게 배운 것 (오늘의 성과)

### 졸음 감지 모델 개발 전체 파이프라인 구성
- **데이터셋 구성**: 'normal' / 'drowsy' 상태 영상 각 200개 이상 수집 및 분류
- **특징 추출 (Feature Engineering)**: dlib의 얼굴 랜드마크 검출을 이용해 EAR(눈 종횡비)·MAR(입 종횡비) 값 추출
  - `EAR < 0.25` 일정 시간 이상 유지 시 졸음으로 판단하는 로직 설계
- **SVM 모델 학습**: scikit-learn의 SVC로 EAR/MAR 피처 기반 이진 분류 모델 훈련
- **실시간 웹캠 모니터링**: 웹캠 피드를 실시간으로 처리하여 졸음 감지 알림 구현

### 데이터 품질 향상 작업
- 불량 샘플 제거 및 데이터 정규화 후 재학습하여 정확도 향상
- dlib의 `shape_predictor_68_face_landmarks.dat` 활용
