## 🚀 새롭게 배운 것
- **Next.js 정적 자산 경로**: `public/` 폴더 내의 자산은 `/` 경로를 통해 직접 접근 가능하다는 점을 다시 한번 확인했습니다.
- **물리적 통신 원리**: 전기 신호(0, 1)가 전압(압력)의 파동 형태로 전달되는 원리를 이해했습니다.
- **측정 방식의 차이**: RS-232는 그라운드 대비 전압을 측정하는 'Single-ended' 방식인 반면, RS-485는 두 선의 전압 차이를 이용하는 'Differential(차동)' 방식이라는 점을 학습했습니다.
- **AI 학습 하이퍼파라미터**: 에폭(반복), 배치(묶음), 학습률(수정폭)의 개념을 '공부' 비유를 통해 정리했습니다. 특히 학습 에러 발생 시 학습률을 가장 먼저 조정해야 한다는 가이드를 수립했습니다.
- **Vercel 배포 문제 해결**: Vercel과 GitHub 연동 끊김 문제를 진단하고 CLI(`vercel link`)를 통해 복구하는 실무 경험을 쌓았습니다.
