## 🚀 새롭게 배운 것 (오늘의 성과)

### YouTube Studio 데이터 자동 수집 파이프라인 설계
- YouTube Data API 또는 Selenium을 통한 YouTube Studio 크롤링 비교 분석
- **Selenium 기반 로그인 자동화**: 구글 계정 로그인 → YouTube Studio → 콘텐츠 탭 → 분석 데이터 추출 흐름 설계
- CTR 데이터를 `video_title - CTR(%)` 테이블 형태로 표준화하는 파싱 로직 작성

### 환경 이슈 확인 및 대응
